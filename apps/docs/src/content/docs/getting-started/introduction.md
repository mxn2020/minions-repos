# Minions Repos

GitHub repo definitions, workflow templates, label sets, and branch protection rules

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-repos/sdk`) — Core library for Node.js
- **Python SDK** (`minions-repos`) — Python bindings
- **CLI Tool** (`@minions-repos/cli`) — Command-line interface
- **Web App** — Interactive playground at [repos.minions.wtf](https://repos.minions.wtf)
